Space Force: The game!

For my final semester at UMD, I made a game for my final project in my Game Programming Class Through this, I was able to learn game development using Unity game engine and learned a new langauge, C#! With the help of two other extremely talented programmers and artists, we were able to implement a whole new physics scheme that features the O'Neil Cylinder! The game has no real win or lose objective, the player explores a space station that goes haywire and droids and robots have gone rogue and attack them! The main objective is to showcase the new physics we were going for and allow the player to just have fun exploring the station with nothing but their laser gun AND grapling hook!
